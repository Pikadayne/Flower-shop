# Thư mục hình ảnh

Thư mục này chứa các hình ảnh sản phẩm của website.

## Cấu trúc đề xuất

```
assets/img/
├── products/          # Hình ảnh sản phẩm
│   ├── product-1.jpg
│   ├── product-2.jpg
│   └── ...
├── banners/           # Hình ảnh banner
│   ├── banner-1.jpg
│   └── banner-2.jpg
└── icons/            # Icon và logo
    └── logo.png
```

## Lưu ý

- Đặt tên file rõ ràng, dễ nhận biết
- Sử dụng định dạng JPG cho ảnh sản phẩm (kích thước nhỏ hơn)
- Sử dụng định dạng PNG cho logo và icon (hỗ trợ trong suốt)
- Tối ưu hóa kích thước file để tăng tốc độ tải trang
